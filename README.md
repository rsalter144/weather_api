# 🌦️ Weather API Project

A Python project that retrieves weather data using the Open-Meteo API and saves it to CSV files.

---

## 📍 Location
- Latitude: 32.0478  
- Longitude: -110.7120  

---

## 📦 Features
- Current temperature
- Humidity
- Feels-like temperature
- Daily highs and lows
- Precipitation data

---

## 🛠️ Technologies
- Python
- requests
- pandas
- Open-Meteo API

---

## ▶️ How to Run

Activate environment:
source myenv/bin/activate

Run script:
python weather_daily.py

---

## 📁 Output Files
- current_weather.csv  
- daily_weather.csv  
- hourly_weather.csv
